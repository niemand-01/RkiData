# 介绍
	- 小程序数据和RKI数据每日更新同步
	- 根据7天内新增感染数比率分别用不同的marker来标记地区
		- >40的标记为红色
		- 20-40的标记为黄色
		- <20的标记为绿色
	- 点击标签可以查看详细信息
	- 在下方有该地区的地名和感染总人数 
	- 一开始加载可能有点慢，大家多多包涵哈

# 数据来源：
